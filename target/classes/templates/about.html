关于简单博客系统

一个基于 Spring Boot 构建的现代化博客平台

项目介绍

简单博客系统是一个使用 Spring Boot 框架开发的现代化博客平台。它提供了简洁的界面和强大的功能，让用户可以轻松地发布、管理和分享文章。

这个项目旨在展示 Spring Boot 在 Web 开发中的应用，包括 MVC 架构、数据持久化、模板引擎等技术的使用。

主要功能

  • 文章发布与编辑
  • 文章列表与详情查看
  • 文章搜索功能
  • 用户管理
  • 响应式设计，支持多设备访问

技术栈

  • 后端: Spring Boot, Spring Data JPA, Spring MVC
  • 前端: Thymeleaf, Bootstrap 5
  • 数据库: H2 Database (内存数据库)
  • 构建工具: Maven
