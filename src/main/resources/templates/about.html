关于这个博客

这是一个基于 Spring Boot + Thymeleaf 的个人写作站点，现在支持 Markdown 创作、分类归档、个人空间、资料加密和个人博客迁移导出。

当前能力

  • 编辑器支持 Markdown、实时预览、文章大纲和本地草稿。
  • 首页固定五个分类：最新、前后端、嵌入式、大数据、项目。
  • 每个注册用户都有自己的个人空间和创作列表。
  • 每个成员都可以把自己的文章一键导出为可直接部署到 GitHub Pages 的静态博客包。
  • 注册完成后即可直接登录并开始创作。

安全处理

  • 登录密码使用 BCrypt 加密存储。
  • QQ 和 GitHub 主页链接以 AES 方式加密后入库。
  • 文章、评论与个人资料的修改和删除均有权限控制。

开源与迁移

站点保留源码仓库入口，毕业后也可以把自己的文章安全迁移到个人博客。
