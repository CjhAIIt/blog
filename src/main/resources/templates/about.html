关于这个博客

这是一个基于 Spring Boot + Thymeleaf 的个人写作站点，现在支持 Markdown 创作、分类归档、个人空间和资料加密。

当前能力

  • 编辑器支持 Markdown、实时预览、文章大纲和本地草稿。
  • 首页固定五个分类：最新、前后端、嵌入式、大数据、项目。
  • 每个注册用户都有自己的个人空间和创作列表。
  • 注册完成后即可直接登录并开始创作。

安全处理

  • 登录密码使用 BCrypt 加密存储。
  • QQ 和 GitHub 主页链接以 AES 方式加密后入库。
  • 个人资料编辑、文章修改和删除都带权限限制。
  • 文章编辑、删除和个人资料修改都收到了权限限制。

作者

站点保留 GitHub 出口，方便继续追踪代码和同步项目资料。
