关于这个博客

这是一个基于 Spring Boot + Thymeleaf 的个人博客系统，现在支持 Markdown 写作、固定分类、个人空间、邮箱验证和资料加密。

当前能力

  • 博客编辑器使用 Markdown，发布后自动渲染。
  • 首页固定五个分类：最新、前后端、嵌入式、大数据、项目。
  • 每个注册用户都有自己的个人空间和创作列表。
  • 注册后必须完成邮箱验证，账号才会启用。

安全处理

  • 登录密码使用 BCrypt 加密存储。
  • QQ 和 GitHub 主页链接以 AES 方式加密后入库。
  • 邮箱验证码带有效期，验证通过后才允许登录。
  • 文章编辑、删除和个人资料修改都收到了权限限制。
