## 🎨 设计系统更新

### CSS 变量系统重构

#### 桌面端 (site.css)
```css
/* 颜色系统 */
--ll-bg: #F8FAFC              /* 从 #f4f7f6 改为中性浅灰白 */
--ll-text: #0F172A            /* 从 #111827 改为更深的文本色 */
--ll-muted: #64748B           /* 从 #617174 改为更清晰的灰色 */
--ll-accent: #3B6FD4          /* 从 #0f766e 青绿改为低饱和蓝 */
--ll-accent-alt: #3B6FD4      /* 统一为同一蓝色，消除双色混乱 */

/* 阴影系统 */
--ll-shadow-soft: 0 2px 8px rgba(15, 23, 42, 0.04), 0 1px 2px rgba(15, 23, 42, 0.03)
--ll-shadow: 0 4px 16px rgba(15, 23, 42, 0.06), 0 1px 4px rgba(15, 23, 42, 0.04)
--ll-shadow-hero: 0 8px 32px rgba(15, 23, 42, 0.08)
/* 阴影大幅降低，增加呼吸感 */
```

#### 移动端 (mobile.css)
```css
--mb-bg: #F1F5F9              /* 从 #eef4f2 改为中性浅灰 */
--mb-text: #0F172A            /* 从 #172033 改为统一的深色 */
--mb-muted: #64748B           /* 从 #62717f 改为统一的灰色 */
--mb-accent: #3B6FD4          /* 从 #0f766e 改为低饱和蓝 */
--mb-shadow: 0 4px 16px rgba(15, 23, 42, 0.06)  /* 大幅降低 */
```

**关键效果：** 单一CSS变量修改自动修复所有引用它的地方（30+处），包括列表项圆点、kicker颜色、侧边栏左边框等。

## 📝 具体改动清单

### 1. CSS 样式修改

| 组件 | 改动 | 效果 |
|------|------|------|
| body 背景 | 去掉渐变，用纯色 | 更干净，减少视觉噪音 |
| topbar 装饰线 | 蓝色渐变 → 中性细线 | 更低调，不抢眼 |
| 导航链接下划线 | 蓝绿渐变 → 单色蓝 | 更简洁，更现代 |
| 主按钮 | 绿到蓝渐变 → 单色蓝 | 去掉花哨感，更专业 |
| 卡片背景 | 多层radial-gradient → 纯白 | 更干净，更有呼吸感 |
| 卡片圆角 | 18px → 12px | 更现代，更精致 |
| 卡片高光 | 删除 inset box-shadow | 更简洁 |
| 卡片封面 | 蓝/蓝绿/蓝紫/蓝灰渐变 → 单色浅灰 | 减少视觉噪音，突出内容 |
| Hero区域 | 多层radial-gradient → 纯白 | 更干净，更有内容感 |
| 文章正文行高 | 2.08 → 1.85 | 更舒适的阅读体验 |
| 文章正文宽度 | 无限制 → max-width: 72ch | 最佳阅读宽度 |
