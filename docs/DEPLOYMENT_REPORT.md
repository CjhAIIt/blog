## 🔒 安全建议

1. **使用SSH密钥认证**
   - 生成密钥：`ssh-keygen -t rsa -b 4096`
   - 复制到服务器：`ssh-copy-id -i ~/.ssh/id_rsa.pub ubuntu@101.35.79.76`

2. **限制SSH访问**
   - 在公网服务器上配置 `/etc/ssh/sshd_config`
   - 只允许特定IP访问

3. **监控隧道连接**
   - 定期检查日志
   - 设置告警规则

4. **定期更新**
   - 更新Nginx：`sudo apt-get update && sudo apt-get upgrade`
   - 更新SSH：`sudo apt-get install openssh-server`

## 🛠️ 故障排查

### 问题：无法访问博客
**解决：**
1. 检查隧道是否运行：`ps aux | grep ssh`
2. 检查公网服务器Nginx：`ssh ubuntu@101.35.79.76 "sudo systemctl status nginx"`
3. 查看错误日志：`ssh ubuntu@101.35.79.76 "sudo tail -f /var/log/nginx/blog-proxy-error.log"`

### 问题：隧道断开
**解决：**
1. 重新建立隧道
2. 检查网络连接
3. 查看SSH日志

### 问题：文件上传失败
**解决：**
1. 检查Nginx配置中的 `client_max_body_size`（已设置为25M）
2. 重启Nginx：`ssh ubuntu@101.35.79.76 "sudo systemctl restart nginx"`
